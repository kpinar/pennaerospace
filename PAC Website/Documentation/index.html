Crafto HTML is a responsive template built on Twitter Bootstrap 3 framework. It's a one-page website ment to present you as a designer (or creative) to the world. Moreover, this theme can be adapted to any kind of creative company. This theme was built using the latest features from Bootstrap 3.1 so it's responsive (mobile first), flat and very nice animated. Also the Parallax effect gives a nice and smooth scrolling experience.

Features

  • Responsive design
  • Built on Boostrap 3.1
  • Valid HTML5 + CSS3
  • Clean design
  • Clean coded
  • PSDs included in package
  • Cross Browser compatible
  • Google Fonts
  • Parallax scrolling
  • Working contact form
  • Unlimited colors by LESS variable
  • Fixed/animated menu
  • CSS3 animations and transitions

B) CSS Files and Structure - top

There are three css files loaded in head.

  1. "bootstrap.css" reset file and template framework.
  2. "font-awesome.min.css" Font Awesome icons font.
  3. "style.css" is the main css file including the elements that were overriden.
  4. "style.less" is the LESS file from wich the main CSS file was generated. So, if you have a LESS compiler, you can use the LESS file to make edits.

The style.css file is separated into sections:

  • /* ---------- General Styles and Components ---------- */
  • /* ---------- Navigation Bar ---------- */
  • /* ---------- Welcome Section ---------- */
  • /* ---------- Portfolio Section ---------- */
  • /* ---------- Testimonials Section ---------- */
  • /* ---------- Team Section ---------- */
  • /* ---------- Services Section ---------- */
  • /* ---------- Blog Section ---------- */
  • /* ---------- Clients Section ---------- */
  • /* ---------- Contact Section ---------- */
  • /* ---------- Footer Section ---------- */
  • /* ---------- CSS Fallbacks for no JS and old browsers ---------- */
  • /* ---------- Media Queries ---------- */

NOTE: If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

C) JavaScript - top

This theme imports more Javascript files located in the "js" folder.

  1. jQuery - Latest jQuery library Located on Google servers
  2. bootstrap.js - java scripts for Bootstrap framework
  3. html5shiv.js - for IE compatibility
  4. jquery.easing.1.3.js - for easing animations
  5. modernizr.custom.47152.js
  6. modernizr-ie.js
  7. waypoints.min.js
  8. jquery.textrotator.min.js
  9. jquery.simple-text-rotator.min.js
  10. grid.js
  11. jquery.nicescroll.min.js
  12. slick.min.js
  13. respond.min.js - for responsive features on old browsers
  14. scripts.js - loaded on the bottom of each pages and contains every little script used for:
    • calling other plugin actions
    • making custom animation and functions
